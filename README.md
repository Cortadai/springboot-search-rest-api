Este es un proyecto de **API REST con Spring Boot** que implementa un motor de búsqueda de productos conectado a una base de datos MySQL. La aplicación permite crear productos y buscarlos mediante palabras clave en el nombre o descripción, demostrando dos enfoques diferentes de consulta: JPQL (Java Persistence Query Language) y SQL nativo. Está organizado en capas siguiendo el patrón MVC (Controller-Service-Repository-Entity), utiliza Spring Data JPA para la persistencia de datos, Lombok para reducir código repetitivo, y viene configurado con Docker Compose para levantar fácilmente la base de datos MySQL y una interfaz web de administración (Adminer). Los endpoints principales son `/api/v1/products/search1` y `/search2` para buscar productos mediante query parameters, y `/create` para crear nuevos productos vía POST, siendo un proyecto educativo ideal para aprender los fundamentos de Spring Boot, JPA/Hibernate y arquitectura de aplicaciones REST en capas.
